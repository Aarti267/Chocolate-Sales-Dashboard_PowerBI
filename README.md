# Chocolate Sales Dashboard | Power BI

## Overview

The Chocolate Sales Dashboard is an interactive Business Intelligence project developed using Power BI to analyze and visualize chocolate sales performance. It provides insights into key business metrics such as revenue, profit, sales trends, product performance, regional sales, and salesperson effectiveness, helping stakeholders make data-driven decisions.

## Key Features

* Interactive dashboard with dynamic filters and slicers
* Revenue, Profit, and Sales KPI cards
* Product-wise and category-wise sales analysis
* Regional sales performance visualization
* Sales representative performance tracking
* Monthly and yearly sales trend analysis
* Top-performing products and regions
* Business insights to identify growth opportunities

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel (Data Source)
* Data Cleaning & Transformation

## Project Outcome

This dashboard enables users to monitor sales performance, identify high-performing products and regions, evaluate sales representative performance, and uncover business trends through interactive visualizations. The project demonstrates practical skills in Power BI, business intelligence, and data-driven decision-making.
